Building Website Procedure

Building websites should be fun

1. Outline

  1. make a simple outline on any writing device
  2. building menu according to it
  3. use this page as a template and build main pages by copying the menu
  4. make the links and menu work so that you can easily navigate between them
  5. write the basic ideas into each page. Now you have the outline of your web.

2. Research

  1. set or refine goals for the website

3. Preparing and writing content

  1. Get images and video clips and sound tracks (It depends. This can be done long time ago or in the last minute or now or even later.)
  2. Write titles and interesting words
  3. avoid long text if possible

4. Plan and design

  1. Optionally, create a sitemap sketch
  2. Optionally, create a wireframe or mock-up
  3. Assess and select technology list, such as languages, frameworks. (Keep learning new stuffs. The world is changing fast!)

5. Coding for each step and design idea

  1. code for main parts of each page
  2. add necessary and interesting features and interactivity
  3. SEO for the website

6. Testing, review and launch

  1. test and debug
  2. proofread and launch
  3. get feedback and make necessary change

7. Maintenance

  1. consider to add user report system
  2. fix bugs
  3. update whenever necessary
